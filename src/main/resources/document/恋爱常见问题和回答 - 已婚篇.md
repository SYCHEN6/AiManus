# 恋爱常见问题和回答 - 已婚篇
#### 婚后感情越来越像“室友”，如何找回亲密感？
“室友感”是婚姻中常见的温水区，源于事务性沟通掩盖了情感交流。可以主动创造“非任务型时间”，比如每周一次专属约会、每天15分钟无手机对话，重新分享感受与梦想，而非只谈孩子、账单和家务。
推荐课程： 《重燃婚姻中的情感连接》

#### 因为钱的问题频繁争吵，怎么破？
金钱争吵背后通常是权力感、安全感与价值观的冲突。建议将“你的钱、我的钱”转化为“我们的钱”视角，设立共同账户与各自自由支配额度，并定期（如每月一次）进行温和的财务沟通，聚焦目标而非互相指责。
推荐课程： 《夫妻财务和谐之道》

#### 有了孩子后夫妻关系变差，如何平衡？
育儿阶段容易将全部精力投向孩子，导致夫妻关系边缘化。核心策略是“夫妻关系优先于亲子关系”——明确分工、保留二人时间、并接受“足够好”的养育标准而非完美主义。父母关系稳定，才是给孩子最好的成长环境。
推荐课程： 《育儿期夫妻关系维护指南》

#### 夫妻之间越来越没话说，是感情出问题了吗？
不一定。长期婚姻中，话题减少有时是生活重合度过高、缺乏新信息输入所致。可以主动引入“新鲜感”：共同学习新技能、一起追剧并讨论、或各自拓展兴趣后再分享。有意识地为关系注入新的话题来源。
推荐课程： 《婚姻中的深度沟通艺术》

#### 遭遇“丧偶式育儿”或家务严重不均，怎么办？
长期失衡会累积怨气。避免用“你从来不…”的指责方式，而是用“具体分工+认可反馈”的方式：明确列出家务与育儿任务清单，协商分配，并在对方做出改变时及时给予正向强化。改变需要过程，重点是建立公平感与合作感。
推荐课程： 《公平分担：婚姻中的合作式分工》
