# 恋爱常见问题和回答 - 恋爱篇
#### 热恋期过后感觉变淡了，是不是不爱了？
热恋期的多巴胺消退是正常生理与心理过程，不代表爱消失。真正的爱会从“激情”转向“亲密与承诺”的稳定阶段。关键不是维持热恋感，而是学会在平淡中创造新的共同体验与深度连接。
推荐课程： 《从热恋到长久的过渡指南》

#### 总是因为小事吵架，怎么打破恶性循环？
反复为小事争吵，往往是因为表层事件下隐藏着未被满足的情感需求（如尊重、安全感、关注）。建议采用“暂停-识别需求-表达请求”的模式：冲突时先暂停，各自识别自己真正在意的是什么，再用“我需要…”代替指责。
推荐课程： 《非暴力沟通在恋爱中的实战应用》

#### 对方总是不回消息，是我不重要吗？
回消息的速度与“重要性”并不完全等同，个体差异很大（如工作性质、独处需求）。但若长期让你感到被忽视，可以约定双方的沟通频率与边界，比如“忙前说一声，忙后主动回”。健康的恋爱是彼此适应，而非单方面猜测。
推荐课程： 《建立健康的恋爱沟通契约》

#### 恋爱中如何保持个人空间，又不让对方觉得疏远？
关键在于“透明化”与“确定性”。你可以提前说明自己需要独处的时间，并明确告诉对方“这段时间后我会来找你”，同时在其他时间里给予高质量陪伴。个人空间不是拒绝，而是为了让彼此以更好的状态相遇。
推荐课程： 《亲密有间：恋爱中的边界感建立》

#### 发现对方有异性好友，该介意吗？
异性好友本身不是问题，关键在于边界是否清晰、互动是否透明、以及你在这段关系中是否感受到尊重。可以坦诚表达自己的感受，而不是命令对方切断关系。健康的信任建立在双方都愿意主动避嫌的基础上。
推荐课程： 《如何处理恋爱中的异性社交边界》
